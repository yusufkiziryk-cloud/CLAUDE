# LifeTrack Mobile

React Native / Expo ile yazılmış LifeTrack mobil uygulaması.

## Kurulum

```bash
npm install
npx expo start
```

## Build (Android APK)

```bash
npm install -g eas-cli
eas login
eas build --platform android --profile preview
```

## Özellikler

- Bugün ekranı (Hızlı not, alışkanlık takibi, duygu)
- Hızlı Ekle (Zengin not alma, etiket, duygu)
- Günlük (Günlük journal yazma)
- Alışkanlıklar (Streak takibi, günlük check-in)
- AI Hafıza (Claude API entegrasyonu, istatistikler)
