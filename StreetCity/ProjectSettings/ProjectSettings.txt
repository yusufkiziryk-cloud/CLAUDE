## Street City: Open World Mobile - Unity Project Settings Rehberi

Unity 2022.3 LTS (Önerilen) veya Unity 2021.3 LTS için aşağıdaki ayarları uygula:

=== PLAYER SETTINGS ===

Edit > Project Settings > Player

[ Android Tab ]

Company Name: YourCompany
Product Name: Street City: Open World Mobile
Version: 1.0.0

Package Name (Bundle Identifier):
  com.yourcompany.streetcity

Minimum API Level: Android 7.0 (API 24)
Target API Level: Android 13 (API 33) - veya Automatic (Highest Installed)

Orientation:
  Default Orientation: Landscape Left (veya Auto Rotation: sadece Landscape Left + Landscape Right işaretle)
  Allow Rotation: SADECE Landscape Left + Landscape Right işaretle

Scripting Backend: IL2CPP (Önerilen - daha iyi performans)
  veya Mono (Daha hızlı build)

Target Architectures:
  ARM64: ✓ (Modern cihazlar için)
  ARMv7: ✓ (Eski cihazlar için opsiyonel)
  x86: ✗ (Android emülatör için gerekli değil)
  x86-64: ✗

Internet Access: Not Required
Write Permission: External (SD Card) - opsiyonel

  Window > Package Manager > Universal RP > Install

  Assets > Create > Rendering > URP Asset (with Universal Renderer)
  Edit > Project Settings > Graphics > Scriptable Render Pipeline Settings > URP Asset ata
